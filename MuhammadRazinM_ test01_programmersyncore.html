1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework ?

Framework adalah sebuah kumpulan kode dasar yang bisa membantu seorang programmer dalam memulai menulis kode. Jadi alasan mengapa setiap bahasa pemrograman memiliki sebuah framework sendiri adalah agar seorang programmer sudah memiliki boilerplate sendiri pada kode yang akan dibuat. Kegunaan dari framework adalah untuk bisa menjadi dasar dalam membuat kode program.

  1. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?
  • If else: berfungsi untuk mengecek nilai pada sebuah variabel, jika bernilai true maka akan dijalankan
  • Switch case: berfungsi untuk mengecek nilai pada sebuah variabel dan mencocokkan dengan case yang ada, jika sesuai maka akan dijalankan. Fungsi utama sama dengan if else namun lebih efektif untuk program yang memiliki banyak kemungkinan
  • While: berfungsi untuk melakukan pengulangan ketika sebuah variabel bernilai true
  • While do: berfungsi untuk melakukan pengulangan dengan cara menjalankan terlebih dahulu pengulangan tersebut dengan kemudian dilakukan pengecekan nilai terhadap variabel
  • For: berfungsi untuk melakukan pengulangan yang biasanya sudah diketahui jumlah pengulangan yang akan dilakukan
  • Foreach: berfungsi untuk melakukan pengulangan dengan membaca nilai pada sebuah array variabel
  1. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?
  • Constructor: sebuah fungsi pertama yang akan dijalankan ketika sebuah class dipanggil
  • Method: sebuah fungsi yang bisa digunakan berulang kali
  • Class: sebuah pengelompokan yang berisi berbagai method di dalamnya
  • Object: sebuah varibel yang bisa digunakan untuk menyimpan class
  • Varibel: penyimpan data atau nilai
  • Parameter: sebuah nilai yang diberikan terhadap suatu method untuk kemudian bisa digunakan
  • Function: sebuah method yang memberikan sebuah return / nilai kembali
  • Procesudre: sebuah method yang tidak memberikan sebuah nilai apa apa setelah melakukan serangkaian kode
